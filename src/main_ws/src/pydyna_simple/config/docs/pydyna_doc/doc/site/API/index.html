create_simulation

create_simulation(*args)

This function is the main API entry-point to create a new Simulation session.

Parameters

  • p3d_or_json_file [optional] (string) : the p3d or json filename containing the hydrodynamics model description to be loaded. If not provided, loads a default hydrodynamics model.

Returns
